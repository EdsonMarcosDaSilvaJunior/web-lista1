Diferença entre Web e Internet

Internet

A internet é uma vasta rede global de redes de computadores interconectados que utilizam um conjunto comum de protocolos para se comunicarem. Em termos simples, é a infraestrutura física e tecnológica que permite a troca de dados entre dispositivos ao redor do mundo. A internet inclui uma ampla gama de serviços e recursos, como e-mails, transmissões de dados, serviços de telefonia VoIP, e muito mais.

Web

A web, ou World Wide Web, é um dos muitos serviços disponíveis na internet. Ela é um sistema de documentos interconectados acessados por meio de navegadores web, usando o protocolo HTTP (Hypertext Transfer Protocol). A web é composta por sites e páginas web que são conectados por hyperlinks e acessados através de URLs (Uniform Resource Locators).
